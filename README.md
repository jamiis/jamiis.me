My personal website
===================

####Run Locally
On Mac OS with node, npm, and homebrew installed

```
git clone --recursive https://github.com/jamiis/jamiis.me.git
cd jamiis.me
npm install
brew install coffee
coffee --compile server/ client/
npm run css
npm run dev
```
